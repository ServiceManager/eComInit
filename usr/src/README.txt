This is the System XVI source tree. It is organised into folders. These are
explained below:

 - agent: Sources to long-running processes that provide System XVI
   functionality.
 - cmake: CMake build-system related files.
 - cmd: Sources to user commands for interacting with System XVI.
 - etc: System configuration files.
 - hdr: Public C/C++ header files.
 = java: Java sources. These are strictly optional components.
 - lib: Library sources.
 - share: System read-only configuration and shared data.
 - test: Testing sources and scripts.
 - tools: Build system tooling.
 - vendor: Source code to other projects used in System XVI.
